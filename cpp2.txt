交互式程序，输入对应输出，
#include <stdio.h>
//platinum 白金
int main(){
	float a,b;	//浮点数
	scanf("%f",&a);	//&，地址符，scanf，读取输入
	b = 1700.0 * a * 14.5833;
	printf("num is %.2f\n",b); //%.2f浮点数只显示小数点后两位
	return 0;
}

预先设定好，整个程序运行过程中不发生改变
运行过程中可能被改变或赋值

数据类型关键字
位，字节和字
位，bit存储0和1
字节，byte，一字节8位，八位字节有256种可能的组合
字，字长不固定，等于n个字节，字长越大，数据转移越快

整数和浮点数
存储方式不同
整数是没有小数部分的数
二进制存储
浮点数，实数，e计数法，E=10的7次方
3.16E7 = 3.16 * 10的7次方，e大小写均可
计算机把小数分为指数部分和小数部分
例7.0=0.7e1
符号位+小数+指数
两个大数相减，浮点数会损失很多精度
浮点数不能表示区间内所有的值，浮点数通常只是实际值的近似值

c语言基本数据类型
int，有符号整型，占用一个机器字长，
int最小取值，-32768~32767
早期为负2的16次方~正2的15次方（一个符号位），现一般为32位，
初始化变量，int a=1;
int a，b=1；，这种格式很糟糕
占位符后面要对应变量，否则会输出一个随机值

八进制和十六进制
0x或0X前缀表示十六进制值
0前缀表示八进制
int x = 0xAFC，y=076;
%d，%o，%x，分别以十进制，八进制，十六进制显示数字
%#o，%#x，%#X，显示各进制数的前缀
